## CSS API

| Variable                  | Default Value          | Description              |
|---------------------------|------------------------|--------------------------|
| `--button__font-size`     | `var(--font-size)`     | The font size            |
| `--button__font-family`   | `var(--font-bold)`     | The font family          |
| `--button__font-weight`   | `normal`               | The weight of the font   |
| `--button__color`         | `var(--color-black)`   | The text color           |
| `--button__text-align`    | `center`               | The aligment of the text |
