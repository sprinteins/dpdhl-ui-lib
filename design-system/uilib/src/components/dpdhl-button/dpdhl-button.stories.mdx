## General CSS APIs

| Variable                    | Default Value          | Description                              |
|-----------------------------|------------------------|------------------------------------------|
| `--button__font-size`       | `var(--font-size)`     | The font size                            |
| `--button__font-family`     | `var(--font-bold)`     | The font family                          |
| `--button__font-weight`     | `normal`               | The weight of the font                   |
| `--button__color`           | `var(--color-black)`   | The text color                           |
| `--button__text-align`      | `center`               | The aligment of the text                 |
| `--button__cursor`          | `pointer`              |                                          |
| `--button__border-radius`   | `var(--border-radius)` |                                          |
| `--button__padding`         | `1rem`                 |                                          |
| `--button__min-width`       | `11rem`                |                                          |
| `--button__line-height`     | `20px`                 |                                          |
| `--button__hover-filter`    | `brightness(115%)`     | The filter set on hover                  |
| `--button__disabled-filter` | `brightness(70%)`      | The filter of disabled button            |
| `--button__active`          | `brightness(85%)`      | The filter set in active state (pushing) |

## Variants' CSS APIs

### Primary 

| Variable                        | Default Value         | Description |
|---------------------------------|-----------------------|-------------|
| `--button--primary__border`     | `none`                |             |
| `--button--primary__color`      | `var(--color-white)`  |             |
| `--button--primary__background` | `var(--color-dhlred)` |             |

### Secondary

| Variable                          | Default Value                     | Description |

| `--button--secondary__color`      | `var(--color-dhlred)`             |             |

### Text Button
