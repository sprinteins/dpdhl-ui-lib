# Get Started

## Install

**yarn:**

```sh
yarn add @sprinteins/dpdhl-uilib
```

**npm:**

```sh
npm install @sprinteins/dpdhl-uilib
```

## Usage

### HTML

> **✅ TIP:** Our HTML example app: [↗ App HTML](https://github.com/sprinteins/dpdhl-ui-lib/tree/main/example-apps/app-html)

Copy the files from the libraries `dist` folder somewhere where
the your html file can access it.

```sh
cp -R node_modules/@sprinteins/dpdhl-ui-lib/dist/* ./public/lib/
```

Then, import it in your html file

```html
<script type="module" src="./lib/index.mjs"></script>
```
